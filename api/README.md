# College - Project Approval

This project just for education purpose, this project used in semester 2 in Widyatama University

# Prerequisites

- Php 8
- Laravel 9
- JWT

# Installation

1. Install dependencies

```shell
php artisan jwt:secret 
```

2. Setup JWT

```shell
php artisan jwt:secret
```

3. Run Server

```shell
php artisan serve
```
